# 特殊变量

1.  `$@`： 代表全部传入参数

2. `$#`: 代表脚本执行时传入的参数总个数

   + `$1、$2、$3...` 依次代表第 1、2、3... 个脚本入参

3. `awk`

   > `echo cat love.txt | awk '{print $2}'`
   >
   > + `awk` 默认以**空格**分割每行文本，分割后的片段依次叫 `$1`、`$2`、`$3`……
   > + `print $2`：打印每行**第二个字段**

4. `bc`：高精度计算器

   > `echo "scale=2; 44/7" | bc`  #  规定保留**2 位小数**，输出6.28

5. `if [ -d "$path" ]`

   | 参数      | 作用                                |
   | --------- | ----------------------------------- |
   | `-d 路径` | 判断是否为存在的目录                |
   | `-f 路径` | 判断是否为存在的普通文件            |
   | `-e 路径` | 判断文件 / 目录是否存在（不管类型） |
   | `-x 路径` | 判断文件是否有可执行权限            |

5. 长度统计

   + `expr length "字符串"`：`expr` 工具自带的长度统计功能，**只统计引号内文字本身的字符数**，不含换行符

     > echo `expr length "I like bash"`  # 11

   + `wc -c`：`wc` = word count，`-c` 统计**总字节数（包含换行符）**

     > echo "I love bash now" | wc -c

   + length：awk 内置函数，计算当前行文本的字符数量，**自动忽略末尾换行**，只统计文字本身

     > echo "I like and love python" | awk '{print length}'

# while

```bash
a=1
while [ $a -le 5 ]
do
    echo "Welcome to Linuxhint $a times"
    a=$(( $a+1 ))
done
```



# IF

```BASH
echo "Enter the name of the file"
# 读取用户键盘输入，存入变量 fname
read fname
# -f 文件测试运算符：判断指定路径是否普通文件存在；
if [ -f "$fname" ]
then
    echo "$fname exist"
else
    echo "$fname does not exist"
fi
```

多参数判断

```bash
# 如果脚本执行的时候没有传入参数

if [[ $# -eq 0 ]]; then
    echo "No argument is given."  # 打印提示：未传入参数
    exit 0                        # 直接结束脚本，退出码0（正常退出）
fi

# $@ 代表全部传入参数
args=("$@")
echo "${args[@]:0:3}"
color=$1
printf "Your favorite color is %s\n" $color
```



# case

```bash
ticket=101
case $ticket in
23)
	# ;;每个分支结束标记，代表跳出 case
    echo "You got the first prize" ;;
503)
    echo "You got the second prize" ;;
101)
    echo "You got the third prize";;
*)
	# *)：通配符，匹配所有其他不满足前面条件的情况
    echo "Sorry, Try again!"
    # 主动退出脚本
    exit 0;; 

# case 语句结束标识（case 倒写
esac
```



# until  与 while

+ `while [ 条件 ]`：条件成立才循环

+ `until [ 条件 ]`：**条件不成立才循环**，刚好和 while 相反

任意 `until [ COND ]`= `while [ ! COND ]`（给条件加逻辑非 `!`）

```bash
x=5
until [ $x -le 0 ]  # while [ ! $x -le 0 ]、while [ $x -gt 0 ]  注意 操作符 !、le前后必须加空格，否则Shell会把[!$x 识别成一整个字符串
do
    sqr=$((x*x))
    echo "The square of $x is $sqr"
    ((x--))
done
```



# function

```bash
x=10
y=15

myFunc()
{
    local x=35
    y=40
    # 函数内任何变量只要没写 local，不管在哪赋值的，都是全局可见
    z=$(( x+y ))
    echo "The sum of $x and $y equal to $z"
}
myFunc # 调用方法，返回：The sum of 35 and 40 equal to 75
# z 定义在函数内部，函数执行完毕后全局不存在 z，$z 为空
echo "The sum of $x and $y equal to $z"  # 调用方法，返回：The sum of 10 and 40 equal to 75
```
