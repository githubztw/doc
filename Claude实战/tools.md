# 文件操作类

| 工具名称 | 核心作用 | 典型使用场景 | 基础用法说明 | 注意事项 |
| -------- | -------- | ------------ | ------------ | -------- |
| Read | 读取本地文本 / 代码文件内容，支持全量读取、按行片段读取 | 查看源码、配置文件、日志、文档内容；分析现有代码逻辑 | 指定**文件路径**，可选补充起始行、结束行，仅读取不修改 | 超大文件会自动截断；二进制文件无法正常解析 |
| Write | 全量写入 / 新建文件，覆盖目标文件全部内容 | 新建代码、配置文件；完整重置文件内容 | 传入「文件路径 + 完整文本内容」，执行整文件写入 | 直接覆盖原有数据，**无撤销**，操作前建议备份 |
| Edit | 局部编辑文件，精准替换指定代码 / 文本片段，不改动其他内容 | 改接口参数、修复 Bug、修改配置项、局部代码优化 | 匹配原有文本块，替换为新内容；支持单行 / 代码块、全局替换 | 依赖文本精准匹配，格式 / 空格不一致会匹配失败 |
| Glob | 基于通配符批量**检索本地文件**（按文件名 / 目录匹配） | 批量查找指定后缀文件、梳理项目目录、筛选资源文件 | 使用 `*`/`**` 通配符，指定检索路径与文件后缀 | 仅匹配**文件名**，不检索文件内部文字 |
| Grep | 按关键词 / 正则**检索文件内部内容**，支持显示行号、上下文 | 全局搜关键字、查报错日志、定位注释、批量检查代码特征 | 指定检索目录 + 关键词 / 正则表达式，可忽略大小写 | 与 Glob 搭配使用：先筛文件，再搜内容 |

# 终端执行类

| 工具名称 | 核心作用 | 典型使用场景 | 基础用法说明 | 注意事项 |
| -------- | -------- | ------------ | ------------ | -------- |
| Bash | 执行 Linux / macOS / WSL 环境下 Shell 终端命令、脚本 | 包安装、Git 操作、编译代码、运行程序、目录管理、脚本执行 | 直接输入标准 Bash 命令，支持管道、重定向、组合指令 | 命令区分大小写；权限不足会执行失败 |
| PowerShell | 执行 Windows 原生 PowerShell 命令、跨平台 PowerShell 脚本 | Windows 系统管理、服务操作、批量文件处理、解析结构化数据 | 输入 PowerShell 标准语法指令，支持对象式操作 | 部分 Linux 命令在此环境不兼容；区分系统版本差异 |

# 代码审查/分析类

| 工具名称 | 核心作用 | 典型使用场景 | 基础用法说明 | 注意事项 |
| -------- | -------- | ------------ | ------------ | -------- |
| Agent | 拆分复杂任务、调度子代理并行 / 串行执行，做多任务协同 | 大型项目重构、多模块开发、全量代码审查、复杂工程拆解 | 定义主任务，拆分多个子任务并分配子代理，设定执行顺序 | 子代理数量过多会消耗资源，合理拆分任务粒度 |
| TaskCreate | 创建结构化任务 / 待办项，记录任务信息与状态 | 开发排期、Bug 跟踪、测试任务、流程待办 | 填写任务标题、描述、优先级、状态，自动生成任务 ID | 任务归属当前会话，跨会话不持久化 |
| TaskUpdate | 修改已有任务的状态、内容、优先级等属性 | 跟进任务进度、标记完成 / 驳回、补充任务备注 | 根据任务 ID，更新对应字段（状态 / 内容 / 优先级） | 必须先通过 TaskList 获取有效任务 ID |
| TaskList | 列出当前所有任务，支持筛选、排序 | 查看任务清单、追溯历史任务、统计完成率 | 可全量查询，也可按「状态 / 优先级」筛选 | 仅展示当前会话内创建的任务 |

# 网络类

| 工具名称 | 核心作用 | 典型使用场景 | 基础用法说明 | 注意事项 |
| -------- | -------- | ------------ | ------------ | -------- |
| WebSearch | 联网搜索网页、文档、技术资料，返回标题、摘要、链接 | 查 API 文档、报错解决方案、技术规范、行业资料 | 输入搜索关键词，引擎返回多条结果摘要与 URL | 仅返回摘要，不会读取网页全文；需要联网 |
| WebFetch | 根据 URL 抓取网页 / 在线文档正文，自动清洗冗余内容 | 精读搜索到的文档、解析在线 PDF、读取技术手册 | 传入目标网页 / 文档 URL，提取纯文本内容 | 无法抓取登录页、验证码页面、强 JS 渲染页面 |

# MCP 类

| 工具名称 | 核心作用 | 典型使用场景 | 基础用法说明 | 注意事项 |
| -------- | -------- | ------------ | ------------ | -------- |
| ListMcpResources | 枚举当前已连接的 MCP 服务，列出所有可用资源 | 初次对接 MCP 服务、盘点外部数据源 / 接口、查看能力清单 | 自动扫描所有 MCP Server，输出资源名称、URI、简介 | 需保证 MCP 服务正常在线、网络连通 |
| ReadMcpResource | 根据资源 URI，读取 MCP 服务提供的具体数据 / 配置 / 文档 | 读取内部配置、设备数据、业务接口返回内容 | 传入 ListMcpResources 获取的资源唯一 URI | 资源权限受限会读取失败；URI 必须严格一致 |

> 补充：MCP = Model Context Protocol，AI 对接外部服务 / 数据源的标准协议。

# 记忆配置类

| 工具名称 | 核心作用 | 典型使用场景 | 基础用法说明 | 注意事项 |
| -------- | -------- | ------------ | ------------ | -------- |
| Skill | 调用预设封装的快捷技能 / 模板、批量流程 | 一键格式化代码、通用脚本、重复工作模板调用 | 直接指定技能名称，自动执行预设逻辑 | 技能需要提前配置，无自定义技能则无法调用 |
| CronCreate | 创建 Cron 表达式风格的定时任务，周期性执行动作 | 定时日志备份、服务巡检、自动同步数据 | 填写任务名 + 标准 Cron 表达式 + 执行内容 | Cron 格式：分 时 日 月 周；时区以系统为准 |
| CronDelete | 根据任务 ID 删除已存在的定时任务 | 废弃过期定时、关停巡检任务 | 传入定时任务 ID 执行删除 | 先通过 CronList 查询有效 ID，避免删错 |
| CronList | 查看全部定时任务，展示周期、状态、下次执行时间 | 运维巡检、管理定时清单、排查任务异常 | 全量列出本机所有 Cron 定时任务 | 仅展示当前创建的任务，系统级计划任务不纳入 |

# 会话控制类

| 工具名称 | 核心作用 | 典型使用场景 | 基础用法说明 | 注意事项 |
| -------- | -------- | ------------ | ------------ | -------- |
| EnterPlanMode | 进入**规划模式**，AI 先输出完整方案，不直接执行 | 架构设计、版本发布、复杂部署、高危操作 | 触发后 AI 梳理步骤、风险、执行方案 | 进入模式后不会自动执行，需人工确认 |
| ExitPlanMode | 退出规划模式，确认执行或驳回修改方案 | 审核方案后，决定开始执行 / 重新规划 | 选择「执行」或「取消」，切换回正常执行模式 | 驳回后需重新梳理方案 |
| AskUserQuestion | 执行中主动暂停，向用户询问信息、确认操作 | 缺失参数、高危删除、多选项分支、二次确认 | 编辑提问内容，等待用户回复后继续流程 | 会中断自动执行，必须人工应答才能继续 |

| -------- | -------- | ------------ | ------------ | -------- |
