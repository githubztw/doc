## 分层依赖（Harness 缰绳）

```plaintext
Domain → Abstractions → Configuration → Infrastructure → Services → Modules → Shell
```

| Layer | 项目 | 职责 | 允许依赖 |
| :--- | :--- | :--- | :--- |
| Domain | `*.Domain` | 实体、值对象、领域枚举 | 无业务项目 |
| Abstractions | `*.Abstractions` | 接口、导航常量、模块契约 | Domain |
| Configuration | `*.Configuration` | Options 绑定、模块配置合并 | Abstractions |
| Infrastructure | `*.Infrastructure` | 日志、持久化桩、HTTP 桩 | Configuration, Abstractions, Domain |
| Services | `*.Services` | 应用服务 / 用例 | Infrastructure 及以下 |
| Modules | `*.Modules.*` | View / ViewModel / Module 注册 | Services 及以下（**禁止** Shell） |
| Shell | `*.Shell` | App、主窗口、主题、全局配置 | 所有模块 + Prism |

## 单模块目录（Prism Module）

```plaintext
Harness.Wpf.Demo.Modules.{Name}/
├── module.settings.json       # 模块默认配置（Harness 要求）
├── {Name}Module.cs              # IModule：RegisterTypes + OnInitialized
├── Configuration/
│   └── {Name}ModuleOptions.cs   # 绑定到 ModuleSettings 节点
├── ViewModels/
├── Views/
└── Assets/                      # 模块私有资源（尽量少用）
```

## Shell 统一配置

`Harness.Wpf.Demo.Shell/appsettings.json`：

```json
{
  "App": { "Title": "Harness WPF Demo", "Theme": "Dark" },
  "Modules": {
    "Dashboard": { "RefreshIntervalSeconds": 30 },
    "Settings": { "ShowAdvanced": false }
  }
}
```

加载顺序：**模块 `module.settings.json` 默认值 → `appsettings.json` 覆盖 → 环境变量（可选）**。

实现位置：`Configuration/ModuleConfigurationComposer.cs`。

## 命名空间

根命名空间：`Harness.Wpf.Demo`。

| 类型 | 约定 | 示例 |
| :--- | :--- | :--- |
| View | `{Name}View` | `DashboardView` |
| ViewModel | `{Name}ViewModel` | `DashboardViewModel` |
| Module | `{Name}Module` | `DashboardModule` |
| Options | `{Name}ModuleOptions` | `DashboardModuleOptions` |
| Service | `I{Name}Service` / `{Name}Service` | `IGreetingService` |
