# 业务模型（Demo）

## 模块边界

| Module | 职责 | 依赖服务 |
| :--- | :--- | :--- |
| Dashboard | 欢迎页、状态摘要 | `IGreetingService`, `IAppStatusService` |
| Settings | 应用标题、主题、模块开关展示 | `ISettingsService` |

## 导航契约

| NavigationKey | 目标 View | 菜单标题 |
| :--- | :--- | :--- |
| `NavigationKeys.Dashboard` | `DashboardView` | 仪表盘 |
| `NavigationKeys.Settings` | `SettingsView` | 设置 |

## 用例（最小 Demo）

1. **UC-01 启动应用**：Shell 加载配置 → 显示主窗体 → 默认导航到 Dashboard
2. **UC-02 刷新问候**：Dashboard 根据 `App:Title` 与模块配置显示问候语
3. **UC-03 查看设置**：Settings 只读展示合并后的 `Modules` 配置（Harness 验证配置链）
