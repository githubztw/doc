# 模块间接口协议（Demo）

## 应用服务接口

| 接口 | 方法 | 说明 |
| :--- | :--- | :--- |
| `IGreetingService` | `string BuildWelcome(string? title)` | 组合欢迎文案 |
| `IAppStatusService` | `AppStatus GetCurrent()` | 当前应用状态摘要 |
| `ISettingsService` | `IReadOnlyList<ModuleSettingEntry> GetModuleSettings()` | 合并后的模块配置展示 |

## 导航 URI（稳定契约）

定义于 `Harness.Wpf.Demo.Abstractions/Navigation/NavigationKeys.cs`：

- `dashboard` → Dashboard 模块主视图
- `settings` → Settings 模块主视图

## 区域名称

- `ContentRegion`：主内容区（Shell 主窗口）

## 事件（可选扩展）

| 事件类型 | 负载 | 发布方 |
| :--- | :--- | :--- |
| `ThemeChangedEvent` | `string Theme` | Settings（未来） |

Demo 阶段以接口注入为主，不强制 EventAggregator。
