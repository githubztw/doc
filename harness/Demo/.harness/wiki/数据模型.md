# 配置与数据模型

## Shell `appsettings.json`

```json
{
  "App": {
    "Title": "Harness WPF Demo",
    "Theme": "Dark"
  },
  "Modules": {
    "Dashboard": {
      "RefreshIntervalSeconds": 30,
      "WelcomeSuffix": "— 驾驭工程样板"
    },
    "Settings": {
      "ShowAdvanced": false
    }
  }
}
```

## 模块默认 `module.settings.json`

每个模块项目根目录一份，键名与 `Modules:{ModuleName}` 节点一致。

| 模块 | 文件路径 | 用途 |
| :--- | :--- | :--- |
| Dashboard | `Modules.Dashboard/module.settings.json` | 刷新间隔、欢迎后缀 |
| Settings | `Modules.Settings/module.settings.json` | 是否显示高级项 |

## 合并规则

1. 读取各模块 `module.settings.json` → `Modules:{Name}`
2. 叠加 Shell `appsettings.json` 的 `Modules` 节点（后者覆盖）
3. 绑定为 `IOptions<TModuleOptions>` 供 ViewModel / Service 使用

## Domain 实体（Demo）

| 类型 | 字段 | 说明 |
| :--- | :--- | :--- |
| `AppStatus` | `Title`, `Theme`, `StartedAt` | 应用运行态摘要 |
| `ModuleSettingEntry` | `ModuleName`, `JsonFragment` | 设置页展示用 |
