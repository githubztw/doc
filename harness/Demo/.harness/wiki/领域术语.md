# 领域术语

| 术语 | 含义 |
| :--- | :--- |
| Shell | 应用程序宿主：App、主窗口、全局主题与配置加载 |
| Module | Prism `IModule`：注册 View/ViewModel 与导航 |
| Region | Prism 区域，如 `ContentRegion` |
| ModuleSettings | 各模块 `module.settings.json` + Shell `appsettings.json` 合并结果 |
| Harness | 驾驭层制品：rules/skills/wiki/automation，用于约束 Agent 生成代码 |
| NavigationKey | 稳定导航 URI，定义于 `Abstractions/Navigation` |
