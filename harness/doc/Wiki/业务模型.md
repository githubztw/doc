# 实体说明

### Product（商品）

|      字段      |   类型    |            描述            |
| :------------: | :-------: | :------------------------: |
| `description`  | `String`  |          商品描述          |
|   `category`   | `String`  |            分类            |
|    `status`    | `Integer` | 状态（0 = 下架，1 = 上架） |
| `priceInCents` | `Integer` |      最低价格（分）🔴       |

------

### ProductSku（商品 SKU）

|      字段      |   类型    |            描述            |
| :------------: | :-------: | :------------------------: |
|      `id`      |  `Long`   |           SKU ID           |
|  `productId`   |  `Long`   |        所属商品 ID         |
|   `skuName`    | `String`  | SKU 名称（如 “红色 - XL”） |
| `priceInCents` | `Integer` |        价格（分）🔴         |
|    `stock`     | `Integer` |          库存数量          |
|    `status`    | `Integer` | 状态（0 = 禁用，1 = 启用） |

------

### Cart（购物车）

|   字段   |  类型  |   描述    |
| :------: | :----: | :-------: |
|   `id`   | `Long` | 购物车 ID |
| `userId` | `Long` |  用户 ID  |

------

### CartItem（购物车项）

|      字段      |   类型    |       描述        |
| :------------: | :-------: | :---------------: |
|      `id`      |  `Long`   |    购物车项 ID    |
|    `cartId`    |  `Long`   |   所属购物车 ID   |
|    `skuId`     |  `Long`   |      SKU ID       |
|   `quantity`   | `Integer` |       数量        |
|  `isSelected`  | `Boolean` |     是否选中      |
| `priceInCents` | `Integer` | 加入时价格（分）🔴 |

------

### Order（订单）

|         字段         |   类型    |     描述      |
| :------------------: | :-------: | :-----------: |
|         `id`         |  `Long`   |    订单 ID    |
|      `orderNo`       | `String`  |    订单号     |
|       `userId`       |  `Long`   |    用户 ID    |
| `totalAmountInCents` | `Integer` | 总金额（分）🔴 |
|       `status`       | `Integer` |   订单状态    |
|       `remark`       | `String`  |     备注      |
