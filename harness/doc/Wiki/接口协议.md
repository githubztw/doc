# 接口协议

## 通用约定

### Base URL

- Development: `http://localhost:8080`
- Production: `https://api.tulingshop.com`

------

### 请求头

|     Header      | Required |    Description     |
| :-------------: | :------: | :----------------: |
| `Content-Type`  |   Yes    | `application/json` |
| `Authorization` |   Yes    |  `Bearer {token}`  |
| `X-Request-Id`  |    No    |    请求追踪 ID     |

------

### 统一响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```
|   Field   |  Type   | Description |
| :-------: | :-----: | :---------: |
|  `code`   | Integer | 业务状态码  |
| `message` | String  |  状态描述   |
|  `data`   |    T    |  业务数据   |

------

### 分页响应

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [],
    "total": 100,
    "page": 1,
    "size": 20
  }
}
```

### 错误码范围

| Range |  Category  |
| :---: | :--------: |
|  200  |    成功    |
|  400  |  参数错误  |
|  401  |   未认证   |
|  403  |   无权限   |
|  404  | 资源不存在 |
|  409  |  业务冲突  |
|  500  | 服务端错误 |

------

## 商品模块 API (核心API)

### `GET /api/v1/products`

获取商品列表（分页）

#### Query Parameters

|   Param    |  Type   | Required |    Description    |
| :--------: | :-----: | :------: | :---------------: |
|   `page`   | Integer |    No    |   页码，默认 1    |
|   `size`   | Integer |    No    | 每页数量，默认 20 |
| `category` | String  |    No    |     分类筛选      |
| `keyword`  | String  |    No    |    关键词搜索     |

#### Response

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [
      {
        "id": 1,
        "name": "商品名称",
        "priceInCents": 9900,
        "category": "电子产品",
        "status": 1
      }
    ],
    "total": 100,
    "page": 1,
    "size":20
  }
}
```
