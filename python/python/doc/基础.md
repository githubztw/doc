**字典** 
字典是无序的键值对集合, 键必须是不可变的类型，如字符串、数字或元组(不能是集合，列表)；
值可以是任何类型。
```python
empty_dict = {}
dct = {'key1':'value1', 'key2':'value2', 'key3':'value3', 'key4':'value4'}

dct['key1'] # 获取字典中的值，不存在时候会返回异常
dct.get('key10') # 获取字典中的值，不存在时候会返回None
dct['key5'] = 'value5' # 添加键值对
dct['key1'] = 'value-one' 修改
dct.pop('key1') # 删除键值对
dct.popitem() # 删除最后一个键值对
del dct['key2'] # 删除 key2 项目
dct.clear() # 删除所有键值对
del dct # 删除字典

dct.copy() # 复制字典,避免原始字典被修改.
dct.keys() # 返回字典中的所有键,返回的是一个dict_keys对象可迭代的，但不支持通过索引访问元素
dct.values() # 返回字典中的所有值,返回的是一个dict_values对象可迭代的，但不支持通过索引访问元素



dct.items() # 返回字典中的所有键值对,返回的是一个dict_items对象可迭代的，但不支持通过索引访问元素
list(dct.items()) # 转换为列表
```

```python
# 定义一个枚举类
from enum import Enum
class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

# 使用枚举类型
print(Color.RED) # Color.RED
print(Color.RED.value) # 1
print(Color(1)) # Color.RED
```


**Zip组装** 
```python
fruits = ['banana']                    
vegetables = ['Tomato']
fruits_and_veges = []
for f, v in zip(fruits, vegetables): # 组装,两个集合长度不一致时候，以小的集合为基准匹配
    fruits_and_veges.append({'fruit':f, 'veg':v})

print(fruits_and_veges) # [{'fruit': 'banana', 'veg': 'Tomato'}]
```
