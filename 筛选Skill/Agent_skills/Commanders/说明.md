7个对应开发生命周期的斩击命令。每个命令都会自动激活正确的技能。

```
 DEFINE          PLAN           BUILD          VERIFY         REVIEW          SHIP
 ┌──────┐      ┌──────┐      ┌──────┐      ┌──────┐      ┌──────┐      ┌──────┐
 │ Idea │ ───▶ │ Spec │ ───▶ │ Code │ ───▶ │ Test │ ───▶ │  QA  │ ───▶ │  Go  │
 │Refine│      │  PRD │      │ Impl │      │Debug │      │ Gate │      │ Live │
 └──────┘      └──────┘      └──────┘      └──────┘      └──────┘      └──────┘
  /spec          /plan          /build        /test         /review       /ship
```

| 你正在做的事   | 指挥             | 关键原则     |
| -------------- | ---------------- | ------------ |
| 定义要建造什么 | `/spec`          | 规范后       |
| 规划如何建造它 | `/plan`          | 小型原子任务 |
| 逐步构建       | `/build`         | 一次切一片   |
| 证明它有效     | `/test`          | 测试就是证明 |
| 合并前的评测   | `/review`        | 改善代码健康 |
| 简化代码       | `/code-simplify` | 清晰胜于聪明 |
| 从发货到生产   | `/ship`          | 越快越安全   |
